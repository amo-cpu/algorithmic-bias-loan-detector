# AI Algorithmic Bias Detector for Loan Approvals

This project simulates an AI-powered loan approval system and evaluates whether
algorithmic decisions introduce bias across socioeconomic groups.

## Features
- AI-driven loan approval model
- Fairness and bias analysis
- Approval rate visualization
- Ethical interpretation of AI decisions
- Fully browser-based (Streamlit)

## Technologies
- Python
- Streamlit
- Pandas / NumPy
- Scikit-learn
- Matplotlib

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
